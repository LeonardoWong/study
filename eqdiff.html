Sezione 1

Introduzione e Definizioni

Un’equazione differenziale è un’equazione in cui l’incognita è una funzione \(y = f(x)\), e nella quale compaiono la funzione stessa e le sue derivate.

Esempio:   \(y' - 4x^2 + 1 = 0\)

Ordine: massimo grado della derivata presente nell’equazione.
Es: \(y'' - 3y' + 5x - 7 = 0\) → equazione di secondo ordine
Curva integrale: il grafico di una soluzione — cioè la curva \(y = f(x)\) nel piano.
Soluzione generale: l’insieme di tutte le infinite soluzioni al variare della costante di integrazione \(+C\).
Soluzione particolare: una singola soluzione della famiglia, determinata da condizioni al contorno.

Esempio fisico — Moto Uniformemente Accelerato

Il sistema cinematico è un caso naturale di equazione differenziale. La tripletta \(\bigl(S(t),\,v(t),\,a(t)\bigr)\) è legata da derivate e integrali:

\[\begin{cases} S(t) = S_0 + v_0\,t + \tfrac{1}{2}\,a\,t^2 \\[4pt] v(t) = S'(t) = v_0 + a\,t \\[4pt] a(t) = v'(t) = S''(t) = a \quad \text{(costante)} \end{cases}\]

La costante \(C\) dipende dalle condizioni iniziali \(S_0\) e \(v_0\).

Condizioni al contorno: l’equazione \(y_0 = f(x_0)\) determina il valore di \(C\).
Se \(x_0 = 0\) si parla di condizione iniziale.

Sezione 2

Problema di Cauchy

Il Problema di Cauchy è la combinazione di: \[\begin{cases} \text{equazione differenziale} & \Longrightarrow \text{soluzione generale} \\[4pt] \text{condizione al contorno} & \Longrightarrow \text{soluzione particolare} \end{cases}\]

Esempio — Problema di Cauchy con \(a(t)=2t\)

Dato \(a(t) = S''(t) = 2t\), trovare \(S(t)\) con le condizioni iniziali \(v_0 = 3\,\text{m/s}\) e \(S_0 = -\tfrac{4}{3}\,\text{m}\).

Passo 1 — velocità

  • \(v(t) = S'(t) = \displaystyle\int 2t\,dt = t^2 + v_0\)
  • Dalla condizione \(v_0 = 3\):   \(v(t) = t^2 + 3\)

Passo 2 — posizione

  • \(S(t) = \displaystyle\int (t^2 + 3)\,dt = \frac{t^3}{3} + 3t + S_0\)
  • Dalla condizione \(S_0 = -\tfrac{4}{3}\):
\[S(t) = \frac{t^3}{3} + 3t - \frac{4}{3}\]

Sezione 3

Equazioni a Variabili Separabili

Un’equazione è a variabili separabili se può essere scritta nella forma: \[y' = g(x)\cdot h(y)\] con \(g(x)\) ed \(h(y)\) contenute rispettivamente solo nelle variabili \(x\) e \(y\).

Metodo risolutivo

Si separano le variabili e si integra membro a membro:

\[\frac{dy}{dx} = g(x)\cdot h(y) \;\Longrightarrow\; \frac{1}{h(y)}\,dy = g(x)\,dx \;\Longrightarrow\; \int \frac{1}{h(y)}\,dy = \int g(x)\,dx\]

Esempio 1

\(y' = 4xy^2\)

  • \(\dfrac{1}{y^2}\,dy = 4x\,dx\)
  • \(\displaystyle\int \frac{1}{y^2}\,dy = \int 4x\,dx \;\Rightarrow\; -\frac{1}{y} = 2x^2 + C\)
\[y = -\dfrac{1}{2x^2 + C}\]

Esempio 2 — con condizione iniziale

\(y' = 6x\,e^{x^2} - 1\),   \(y(0) = 4\)

  • \(\displaystyle\int dy = \int \!\bigl(6x\,e^{x^2} - 1\bigr)\,dx\)
  • \(y = 3\displaystyle\int\! 2x\,e^{x^2}\,dx - x = 3e^{x^2} - x + C\)
  • Condizione iniziale: \(4 = 3e^0 - 0 + C \;\Rightarrow\; C = 1\)
\[y = 3e^{x^2} - x + 1\]

Esempio 3 — integrazione per parti

\(e^x y' - x = 0\),   \(y(1) = \dfrac{1}{e}\)

  • \(y' = \dfrac{x}{e^x} = x\,e^{-x}\)
  • Integrazione per parti \(\bigl(u = x,\; dv = e^{-x}dx\bigr)\): \[y = \int x\,e^{-x}\,dx = -x\,e^{-x} - e^{-x} + C = -\frac{x+1}{e^x} + C\]
  • Condizione: \(\;\dfrac{1}{e} = -\dfrac{2}{e} + C \;\Rightarrow\; C = \dfrac{3}{e}\)
\[y = -\frac{x+1}{e^x} + \frac{3}{e}\]

Sezione 4

Equazioni Lineari del Primo Ordine

Forma generale: \[y' = a(x)\,y + b(x)\] È lineare perché \(y\) ha potenza 1. Le funzioni \(a(x)\) e \(b(x)\) sono continue nell’intervallo in cui si sta lavorando.

A — Omogenee  (senza termine noto)

Se \(b(x) = 0\), l’equazione diventa \(y' = a(x)\,y\) — un’equazione a variabili separabili.

Esempio — Omogenea

\(y' = \ln x \cdot y\),   con \(a(x) = \ln x,\; b(x) = 0\)

  • \(\dfrac{1}{y}\,dy = \ln x\,dx\)
  • \(\displaystyle\int \frac{1}{y}\,dy = \int \ln x\,dx\)
  • \(\ln|y| = x\ln x - x + C\)   \(\bigl(\int \ln x\,dx = x\ln x - x\bigr)\)
  • \(y = e^{x\ln x - x + C} = e^C \cdot e^{x\ln x} \cdot e^{-x}\)
\[y = k\, e^{x(\ln|x|-1)} = k\,x^x\,e^{-x} \quad (k = e^C)\]

B — Complete  (con termine noto)

Se \(b(x) \neq 0\). Si calcola la primitiva di \(a(x)\): \[A(x) = \int a(x)\,dx \qquad \text{(posta } c = 0\text{)}\]

Derivazione della formula risolutiva

  • Moltiplicare per \(e^{-A(x)}\): \[e^{-A(x)}\bigl(y' - a(x)y\bigr) = b(x)\,e^{-A(x)}\]
  • Poiché \(-A'(x) = -a(x)\), si ha \(\;D\!\bigl(e^{-A(x)}\bigr) = -a(x)\,e^{-A(x)}\), quindi: \[e^{-A(x)}y' + D\!\bigl(e^{-A(x)}\bigr)\,y = D\!\bigl(e^{-A(x)}\,y\bigr)\]
  • \[D\!\bigl(e^{-A(x)}\,y\bigr) = b(x)\,e^{-A(x)}\]
  • Integrando entrambi i membri: \[e^{-A(x)}\,y = \int b(x)\,e^{-A(x)}\,dx\]
Formula risolutiva \[y = e^{A(x)}\!\left[\int b(x)\,e^{-A(x)}\,dx + C\right]\]

Esempio — Equazione Completa

\(y' + \dfrac{y}{x} = x\),   \(x > 0\)   →   \(a(x) = -\dfrac{1}{x},\quad b(x) = x\)

  • \(A(x) = \displaystyle\int -\frac{1}{x}\,dx = -\ln x\)
  • \(e^{-A(x)} = e^{\ln x} = x\)
  • Moltiplicare l’equazione per \(x\): \[xy' + y = x^2\]
  • Si riconosce la derivata di un prodotto: \[D(x\,y) = x^2\]
  • Integrare: \[x\,y = \int x^2\,dx = \frac{x^3}{3} + C\]
\[y = \frac{x^2}{3} + \frac{C}{x}\]

Sezione 5

Verifica di un’Equazione Differenziale

Per verificare che una funzione soddisfa un’equazione differenziale, si calcolano le derivate necessarie, le si sostituiscono nell’equazione e si verifica che si ottiene un’identità \((0 = 0)\).

Esempio 1 — Verifica algebrica

Verificare che \(\;y = \dfrac{\ln x}{x}\;\) soddisfa l’equazione differenziale:

\[x^2\,y'' + x\,y' + \frac{2}{x} = y\]

Calcolo delle derivate

  • \(y' = \dfrac{\dfrac{1}{x}\cdot x - \ln x}{x^2} = \dfrac{1 - \ln x}{x^2}\)
  • \(y'' = \dfrac{-\dfrac{1}{x}\cdot x^2 - (1-\ln x)\cdot 2x}{x^4} = \dfrac{-x - 2x + 2x\ln x}{x^4} = \dfrac{2\ln x - 3}{x^3}\)

Sostituzione nell’equazione

  • \(x^2 \cdot \dfrac{2\ln x - 3}{x^3} \;+\; x\cdot\dfrac{1 - \ln x}{x^2} \;+\; \dfrac{2}{x}\)
  • \(= \dfrac{2\ln x - 3}{x} + \dfrac{1 - \ln x}{x} + \dfrac{2}{x}\)
  • \(= \dfrac{2\ln x - 3 + 1 - \ln x + 2}{x} = \dfrac{\ln x}{x} = y \quad\checkmark\)
0 = 0  →  VERIFICA ✓

Esempio 2 — Circuito elettrico (Legge delle Maglie)

In un circuito RL, la Legge delle Maglie fornisce:

\[\mathcal{E}^0 - L\,i'(t) - R\,i(t) = 0\]

Questa è un’equazione differenziale: contiene la funzione \(i(t)\) e la sua derivata \(i'(t)\); l’incognita è la funzione \(i(t)\).

Verificare che \(\;i(t) = \dfrac{\mathcal{E}^0}{R}\!\left(1 - e^{-\frac{R}{L}t}\right)\) soddisfa l’equazione.

Derivata di \(i(t)\)

  • \(i'(t) = \dfrac{\mathcal{E}^0}{R}\cdot\dfrac{R}{L}\cdot e^{-\frac{R}{L}t} = \dfrac{\mathcal{E}^0}{L}\,e^{-\frac{R}{L}t}\)

Sostituzione

  • \(\mathcal{E}^0 \;-\; L\cdot\dfrac{\mathcal{E}^0}{L}\,e^{-\frac{R}{L}t} \;-\; R\cdot\dfrac{\mathcal{E}^0}{R}\!\left(1-e^{-\frac{R}{L}t}\right)\)
  • \(= \mathcal{E}^0 - \mathcal{E}^0\,e^{-\frac{R}{L}t} - \mathcal{E}^0\!\left(1 - e^{-\frac{R}{L}t}\right)\)
  • \(= \mathcal{E}^0\!\left(1 - e^{-\frac{R}{L}t} - 1 + e^{-\frac{R}{L}t}\right) = 0\quad\checkmark\)
0 = 0  →  VERIFICA ✓
