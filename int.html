01

Primitiva e Tabella degli Integrali

Definizione — Primitiva

\(F(x)\) è una primitiva di \(f(x)\) in \([a,b]\) se \(F\) è derivabile in \([a,b]\) e \(F'(x) = f(x)\) per ogni \(x \in [a,b]\).

L'integrale indefinito raccoglie tutte le primitive di \(f\): differiscono solo per una costante additiva \(c \in \mathbb{R}\).

\[\int f(x)\,dx = F(x) + c\]
Linearità dell'integrale

\(\displaystyle\int [\alpha\, f(x) + \beta\, g(x)]\,dx = \alpha\int f(x)\,dx + \beta\int g(x)\,dx\)

Tavola delle formule fondamentali

Ogni formula di integrazione è il contrario della derivazione corrispondente.

Derivata \(F'(x)\)Integrale \(\int f\,dx\)
\(D[x^n] = nx^{n-1}\)↔\(\displaystyle\int x^n\,dx = \dfrac{x^{n+1}}{n+1} + c \quad (n \neq -1)\)
\(D[e^x] = e^x\)↔\(\displaystyle\int e^x\,dx = e^x + c\)
\(D[\ln|x|] = \dfrac{1}{x}\)↔\(\displaystyle\int \frac{1}{x}\,dx = \ln|x| + c\)
\(D[\sin x] = \cos x\)↔\(\displaystyle\int \cos x\,dx = \sin x + c\)
\(D[\cos x] = -\sin x\)↔\(\displaystyle\int \sin x\,dx = -\cos x + c\)
\(D[\tan x] = \dfrac{1}{\cos^2 x}\)↔\(\displaystyle\int \frac{1}{\cos^2 x}\,dx = \tan x + c\)
\(D[\cot x] = -\dfrac{1}{\sin^2 x}\)↔\(\displaystyle\int \frac{1}{\sin^2 x}\,dx = -\cot x + c\)
\(D[\arctan x] = \dfrac{1}{1+x^2}\)↔\(\displaystyle\int \frac{1}{1+x^2}\,dx = \arctan x + c\)
\(D[\arcsin x] = \dfrac{1}{\sqrt{1-x^2}}\)↔\(\displaystyle\int \frac{1}{\sqrt{1-x^2}}\,dx = \arcsin x + c\)
\(D[a^x] = a^x \ln a\)↔\(\displaystyle\int a^x\,dx = \frac{a^x}{\ln a} + c \quad (a>0,\,a\neq1)\)
\(D[\log_a|x|] = \dfrac{1}{x\ln a}\)↔\(\displaystyle\int \frac{1}{x}\,dx = \frac{\ln|x|}{\ln a} \cdot \ln a = \ln|x| + c\)
Esempio — Polinomio

\(\displaystyle\int (3x^2 - 5x^4 + 4x^4 - 3)\,dx = x^3 - \tfrac{5}{4}x^4 + \tfrac{4}{5}x^5 - 3x + c\)

02

Integrali Composti

Quando l'argomento della funzione è a sua volta una funzione \(g(x)\), si applicano le formule in versione composta: basta moltiplicare per la derivata interna \(g'(x)\).

Schema generale

\(\displaystyle\int [g(x)]^n \cdot g'(x)\,dx = \frac{[g(x)]^{n+1}}{n+1} + c \qquad (n \neq -1)\)

Forma compostaRisultato
\(\displaystyle\int [g(x)]^n g'(x)\,dx\)→\(\dfrac{[g(x)]^{n+1}}{n+1}+c\)
\(\displaystyle\int e^{g(x)} g'(x)\,dx\)→\(e^{g(x)}+c\)
\(\displaystyle\int \frac{g'(x)}{g(x)}\,dx\)→\(\ln|g(x)|+c\)
\(\displaystyle\int \cos[g(x)]\cdot g'(x)\,dx\)→\(\sin[g(x)]+c\)
\(\displaystyle\int \sin[g(x)]\cdot g'(x)\,dx\)→\(-\cos[g(x)]+c\)
\(\displaystyle\int \frac{g'(x)}{1+[g(x)]^2}\,dx\)→\(\arctan[g(x)]+c\)
Attenzione

Se \(g'(x)\) non compare già nell'integrando, occorre aggiustarla moltiplicando e dividendo per la costante opportuna. Non si può aggiustare una \(x\) mancante — in quel caso si usa la sostituzione.

Esempi tipici

Esempio 1 — Potenza composta

\(\displaystyle\int 2x(x^2-1)^5\,dx = \frac{(x^2-1)^6}{6} + c\)  [poiché \(D(x^2-1)=2x\)]

Esempio 2 — Logaritmo composto

\(\displaystyle\int \frac{3x^2+4x+1}{x^3+2x^2+x}\,dx = \ln|x^3+2x^2+x| + c\)

Esempio 3 — Esponenziale composto

\(\displaystyle\int 3e^{-3x}\,dx = -e^{-3x} + c\)  (con aggiustamento del segno)

Esempio 4 — Radice al denominatore

\(\displaystyle\int \frac{e^{1/x}}{x^2}\,dx = -e^{1/x} + c\)  [poiché \(D(x^{-1}) = -x^{-2}\)]

03

Integrazione per Sostituzione

Quando la funzione composta non presenta il fattore \(g'(x)\) in modo diretto, conviene cambiare variabile introducendo \(t = g(x)\).

Metodo

Si pone \(x = \varphi(t)\), da cui \(dx = \varphi'(t)\,dt\). Si sostituisce tutto, si integra in \(t\), e infine si ritorna alla variabile \(x\).

  1. Scegliere la sostituzione \(t = g(x)\) o \(x = \varphi(t)\) in modo che l'integrale diventi più semplice.
  2. Calcolare il differenziale: \(dt = g'(x)\,dx\) oppure \(dx = \varphi'(t)\,dt\).
  3. Sostituire sia la funzione che il differenziale nell'integrale.
  4. Integrare rispetto a \(t\).
  5. Tornare alla variabile \(x\) sostituendo \(t = g(x)\).

Sostituzioni classiche da ricordare

Presenza di \(\sqrt{x}\)

Poni \(t = \sqrt{x}\), cioè \(x = t^2\), \(dx = 2t\,dt\)

Presenza di \(\sqrt{ax+b}\)

Poni \(t = \sqrt{ax+b}\), \(x = \frac{t^2-b}{a}\), \(dx = \frac{2t}{a}\,dt\)

Presenza di \(e^x\)

Poni \(t = e^x\), \(x = \ln t\), \(dx = \frac{1}{t}\,dt\)

Presenza di \(\ln x\)

Poni \(t = \ln x\), \(x = e^t\), \(dx = e^t\,dt\)

Esempio — \(\int \frac{1}{1+\sqrt{x}}\,dx\)

Pongo \(t = \sqrt{x}\), \(x = t^2\), \(dx = 2t\,dt\):

\(\displaystyle\int \frac{2t}{1+t}\,dt = 2\int \frac{t+1-1}{1+t}\,dt = 2\int\!\left(1 - \frac{1}{1+t}\right)dt = 2t - 2\ln|1+t| + c = 2\sqrt{x} - \ln(1+\sqrt{x})^2 + c\)

Esempio — \(\int \frac{1}{e^x + e^{-x}}\,dx\)

Pongo \(t = e^x\), \(dx = \frac{1}{t}\,dt\):

\(\displaystyle\int \frac{1}{t+\frac{1}{t}}\cdot\frac{1}{t}\,dt = \int\frac{1}{t^2+1}\,dt = \arctan t + c = \arctan(e^x) + c\)

04

Integrazione per Parti

Deriva dalla regola del prodotto per le derivate: si usa quando l'integrando è un prodotto di due funzioni di tipo diverso.

\[\int f(x)\,g'(x)\,dx = f(x)\,g(x) - \int f'(x)\,g(x)\,dx\]
Regola pratica — ILATE

Scegli \(f(x)\) (da derivare) seguendo l'ordine di priorità: Inversa trigonometrica → Logaritmica → Algebrica (polinomio) → Trigonometrica → Esponenziale.
La funzione \(g'(x)\) è quella rimanente (da integrare).

Esempi fondamentali

Esempio 1 — \(\int x\ln x\,dx\)

\(f(x) = \ln x\) (logaritmica, ha priorità), \(g'(x) = x \Rightarrow g(x) = \frac{x^2}{2}\)

\(\displaystyle = \frac{x^2}{2}\ln x - \int \frac{x^2}{2}\cdot\frac{1}{x}\,dx = \frac{x^2}{2}\ln x - \frac{x^2}{4} + c = \frac{x^2}{4}(2\ln x - 1) + c\)

Esempio 2 — \(\int x\sin x\,dx\)

\(f(x) = x\), \(g'(x) = \sin x \Rightarrow g(x) = -\cos x\)

\(\displaystyle = -x\cos x + \int \cos x\,dx = -x\cos x + \sin x + c\)

Esempio 3 — \(\int x^2 e^x\,dx\) (doppia applicazione)

\(\displaystyle = x^2 e^x - 2\int xe^x\,dx = x^2 e^x - 2\left(xe^x - e^x\right) + c = e^x(x^2 - 2x + 2) + c\)

Esempio 4 — \(\int \sin^2 x\,dx\) (integrale ciclico)

Integro per parti due volte: l'integrale compare da entrambi i lati e si risolve algebricamente.

\(\displaystyle\int\sin^2 x\,dx = -\sin x\cos x + \int\cos^2 x\,dx = -\sin x\cos x + \int(1-\sin^2 x)\,dx\)

\(\displaystyle 2\int\sin^2 x\,dx = x - \sin x\cos x \implies \int\sin^2 x\,dx = \frac{x - \sin x\cos x}{2} + c\)

Esempio 5 — \(\int \ln x\,dx\)

\(f(x) = \ln x\), \(g'(x) = 1 \Rightarrow g(x) = x\)

\(\displaystyle = x\ln x - \int x\cdot\frac{1}{x}\,dx = x\ln x - x + c = x(\ln x - 1) + c\)

05

Funzioni Fratte

L'integrale di una funzione razionale \(\frac{P(x)}{Q(x)}\) si affronta con tecniche diverse a seconda dei gradi di \(P\) e \(Q\) e della fattorizzazione del denominatore.

Caso 1 — Grado numeratore ≥ grado denominatore

Si esegue prima la divisione polinomiale e si integra il risultato.

Esempio

\(\displaystyle\int \frac{x^3+2x^2+x+1}{x^2+1}\,dx\)

Divisione: \(\frac{x^3+2x^2+x+1}{x^2+1} = x+2 - \frac{1}{x^2+1}\)

\(\displaystyle = \frac{x^2}{2} + 2x - \arctan x + c\)

Caso 2 — Grado numeratore < grado denominatore, \(\Delta > 0\)

Il denominatore si fattorizza in radici reali distinte → si usano i fratti parziali.

Decomposizione in fratti semplici

\(\displaystyle\frac{P(x)}{(x-a)(x-b)} = \frac{A}{x-a} + \frac{B}{x-b}\)

Si determinano \(A,B\) uguagliando i numeratori e risolvendo il sistema.

Esempio — \(\int \frac{5x-1}{x^2-x-2}\,dx\)

\(x^2-x-2=(x-2)(x+1)\)  →  \(\frac{5x-1}{(x-2)(x+1)} = \frac{3}{x-2}+\frac{2}{x+1}\)

\(\displaystyle = 3\ln|x-2| + 2\ln|x+1| + c = \ln|(x-2)^3(x+1)^2| + c\)

Caso 3 — \(\Delta = 0\) (radice doppia)

Decomposizione con radice doppia

\(\displaystyle\frac{P(x)}{(x-a)^2} = \frac{A}{x-a} + \frac{B}{(x-a)^2}\)

Esempio — \(\int \frac{2x-3}{(x-2)^2}\,dx\)

\(\frac{2x-3}{(x-2)^2} = \frac{2}{x-2} + \frac{1}{(x-2)^2}\)

\(\displaystyle = 2\ln|x-2| - \frac{1}{x-2} + c\)

Caso 4 — \(\Delta < 0\) (nessuna radice reale)

Il denominatore quadratico si porta nella forma \((x+p)^2 + q^2\) e si usano arcotangente e logaritmo.

Esempio — \(\int \frac{1}{x^2+2x+2}\,dx\)

\(x^2+2x+2 = (x+1)^2+1\)

\(\displaystyle = \arctan(x+1) + c\)

Denominatori di grado > 2

Completamente fattorizzabili

\(\displaystyle\frac{x-1}{(x-2)^3} \to \frac{A}{x-2}+\frac{B}{(x-2)^2}+\frac{C}{(x-2)^3}\)

Con fattore irriducibile

\(\displaystyle\frac{x-3}{(x+2)(x^2+1)} \to \frac{A}{x+2}+\frac{Bx+C}{x^2+1}\)

Integrale Definito e Calcolo delle Aree

Definizione — Integrale di Riemann

Data \(f\) continua in \([a,b]\), si divide l'intervallo in \(n\) parti uguali di ampiezza \(\Delta x = \frac{b-a}{n}\). Si costruiscono somme di rettangoli con altezza pari al minimo (\(A_1\)) e al massimo (\(A_2\)) su ogni sotto-intervallo:

\(\displaystyle A_1 = \Delta x \sum_{i=1}^n m_i \leq A_{tot} \leq A_2 = \Delta x \sum_{i=1}^n M_i\)

Teorema dei Carabinieri (per gli integrali)

\(A_1 \leq A_{tot} \leq A_2\) e, per il Teorema di Weierstrass e dei valori intermedi:

\(\displaystyle\lim_{\Delta x \to 0} A_1 = \lim_{\Delta x \to 0} A_2 = \int_a^b f(x)\,dx\)

Proprietà dell'integrale definito

  • \(\displaystyle\int_a^a f(x)\,dx = 0\)
  • \(\displaystyle\int_a^b f\,dx = -\int_b^a f\,dx\)
  • \(\displaystyle\int_a^b [f+g]\,dx = \int_a^b f\,dx + \int_a^b g\,dx\)
  • \(\displaystyle\int_a^b f\,dx = \int_a^c f\,dx + \int_c^b f\,dx \qquad (a
  • Se \(f(x) \geq 0\) in \([a,b]\) allora \(\int_a^b f\,dx \geq 0\)

Calcolo di Aree

Attenzione al segno

L'area è sempre positiva. Se \(f(x) < 0\) su parte dell'intervallo, l'integrale definito può essere negativo: l'area si calcola come \(A = \left|\int_a^b f(x)\,dx\right|\) oppure spezzando l'intervallo.

Area tra due curve

Se \(f(x) \geq g(x)\) in \([a,b]\):  \(\displaystyle A = \int_a^b [f(x) - g(x)]\,dx\)

Se la funzione cambia segno in \(c \in (a,b)\) con \(f(c)=0\):

\[A = \left|\int_a^c f(x)\,dx\right| + \left|\int_c^b f(x)\,dx\right|\]
Esempio — Area di \(\sin x\) in \([0, 2\pi]\)

\(\displaystyle\int_0^{2\pi}\sin x\,dx = 0\) (le aree si cancellano)

Area effettiva: \(\displaystyle A = \int_0^\pi \sin x\,dx + \left|\int_\pi^{2\pi}\sin x\,dx\right| = 2 + 2 = 4\)

07

Teorema Fondamentale del Calcolo Integrale

Teorema Fondamentale — Enunciato

Se \(f\) è continua in \([a,b]\) e \(F\) è una sua primitiva, allora:

\(\displaystyle\int_a^b f(x)\,dx = F(b) - F(a) =: \Big[F(x)\Big]_a^b\)

  1. Calcolare la famiglia di primitive \(F(x)\) dell'integrando (l'integrale indefinito).
  2. Valutare \(F(b)\) e \(F(a)\).
  3. Calcolare la differenza \(F(b) - F(a)\). La costante \(c\) si annulla.
Esempio

\(\displaystyle\int_1^2 x^2\,dt = \left[\frac{x^3}{3}\right]_1^2 = \frac{8}{3} - \frac{1}{3} = \frac{7}{3}\)

Integrazione per parti con estremi

\[\int_a^b f(x)g'(x)\,dx = \Big[f(x)g(x)\Big]_a^b - \int_a^b f'(x)g(x)\,dx\]

Sostituzione nell'integrale definito

Cambio di variabile — estremi

Con la sostituzione \(x = \varphi(t)\) occorre anche cambiare gli estremi: se \(x=a \Rightarrow t = \varphi^{-1}(a)\), se \(x=b \Rightarrow t = \varphi^{-1}(b)\). Non si torna alla variabile originale.

Teorema della Media Integrale

Teorema della Media

Se \(f\) è continua in \([a,b]\), allora esiste almeno un punto \(c \in (a,b)\) tale che:

\(\displaystyle f(c) = \frac{1}{b-a}\int_a^b f(x)\,dx\)

Il valore \(\displaystyle\mu = \frac{1}{b-a}\int_a^b f(x)\,dx\) si chiama valor medio (o media integrale) di \(f\) su \([a,b]\). Geometricamente, è l'altezza del rettangolo con base \([a,b]\) che ha la stessa area della regione sotto il grafico di \(f\).

Dimostrazione (schema)

Per il Teorema di Weierstrass esistono \(m = \min f\) e \(M = \max f\) in \([a,b]\). Integrando la disuguaglianza \(m \leq f(x) \leq M\) su \([a,b]\):

\(\displaystyle m(b-a) \leq \int_a^b f\,dx \leq M(b-a) \implies m \leq \frac{\int_a^b f\,dx}{b-a} \leq M\)

Per il Teorema dei valori intermedi, \(f\) assume quel valore in qualche \(c \in (a,b)\).

Esempio — Verificare il teorema per \(y = \ln(x+3)\) su \([-2,0]\)

\(\displaystyle\int_{-2}^0 \ln(x+3)\,dx = \Big[x\ln(x+3)\Big]_{-2}^0 - \int_{-2}^0\frac{x}{x+3}\,dx = -2\ln 1 - (0) - \int_{-2}^0\!\left(1-\frac{3}{x+3}\right)\!dx\)

\(\displaystyle = 3\ln 3 - 2\)

Valor medio: \(\mu = \frac{3\ln 3-2}{2}\). Si trova \(c\) da \(\ln(c+3) = \mu\).

09

Funzione Integrale

Definizione

\(\displaystyle G(x) = \int_a^x f(t)\,dt\)

Si usa \(t\) (variabile muta) per non confondere con l'estremo superiore variabile \(x\).

Derivata della funzione integrale

Se \(f\) è continua, allora \(G'(x) = f(x)\). In altri termini: derivare l'integrale rispetto all'estremo superiore restituisce l'integrando valutato in quel punto.

\(\textbf{Caso con funzione composta:}\quad\) se \(\displaystyle G(x) = \int_a^{g(x)} f(t)\,dt\) allora \(G'(x) = f(g(x))\cdot g'(x)\)

Esempi

\(\displaystyle G(x) = \int_2^x e^t\,dt \implies G'(x) = e^x\)

\(\displaystyle G(x) = \int_2^x t\,dt \implies G'(x) = x\)

\(\displaystyle G(x) = \int_2^{3x} h(t)\,dt \implies G'(x) = h(3x)\cdot 3\)

Lo studio della funzione integrale (monotonia, concavità, ecc.) si fa studiando \(G'(x) = f(x)\) e \(G''(x) = f'(x)\), esattamente come per qualsiasi altra funzione.

Volumi di Rotazione

Ruotando il grafico di \(f(x)\) attorno ad un asse si genera un solido di rotazione. Il volume si calcola integrando le aree delle sezioni circolari.

Rotazione attorno all'asse x

Ogni sezione è un cerchio di raggio \(f(x)\) e area \(\pi [f(x)]^2\):

\[V = \pi\int_a^b [f(x)]^2\,dx\]
Rotazione attorno all'asse y

Si inverte la funzione: \(x = f(y)\). Ogni sezione ha raggio \(f(y)\):

\[V = \pi\int_a^b [f(y)]^2\,dy\]
Invertibilità

Per ruotare attorno all'asse \(y\), la funzione deve essere invertibile nell'intervallo considerato. Verificare che il dominio garantisca la biettività.

Metodo dei gusci cilindrici

Alternativa alla rotazione attorno all'asse \(y\) senza dover invertire la funzione:

\[V = 2\pi\int_a^b x\,f(x)\,dx\]

Ogni guscio cilindrico ha raggio \(x\), altezza \(f(x)\) e spessore \(dx\).

Esempio — \(y = x^2\) ruotata attorno all'asse \(y\), \(y \in [1,4]\)

Funzione inversa: \(x = \sqrt{y}\), quindi \([f(y)]^2 = y\)

\(\displaystyle V = \pi\int_1^4 y\,dy = \pi\left[\frac{y^2}{2}\right]_1^4 = \pi\left(\frac{16}{2} - \frac{1}{2}\right) = \frac{15\pi}{2}\)

Integrale Improprio

Si parla di integrale improprio quando la funzione ha una discontinuità nell'intervallo oppure quando uno o entrambi gli estremi di integrazione sono infiniti.

Estremo infinito

\(\displaystyle\int_a^{+\infty} f\,dx = \lim_{c\to+\infty}\int_a^c f\,dx\)

Discontinuità in b

\(\displaystyle\int_a^b f\,dx = \lim_{c\to b^-}\int_a^c f\,dx\)

Discontinuità interna

\(\displaystyle\int_a^b = \int_a^{x_0} + \int_{x_0}^b\)

(con doppio limite)

Convergenza e Divergenza

Se il limite esiste finito → l'integrale converge.

Se il limite è \(\pm\infty\) o non esiste → l'integrale diverge.

Esempio 1 — Discontinuità in 0: \(\int_{-2}^{0}\frac{1}{\sqrt{-x}}\,dx\)

Discontinuità di 2ª specie in \(x=0\). Prendo \(c \in (-2,0)\), \(c \to 0^-\):

\(\displaystyle\lim_{c\to 0^-}\int_{-2}^c (-x)^{-1/2}\,dx = \lim_{c\to 0^-}\left[-2\sqrt{-x}\right]_{-2}^c = \lim_{c\to 0^-}\left(-2\sqrt{-c}+2\sqrt{2}\right) = 2\sqrt{2}\)

→ Converge a \(2\sqrt{2}\).

Esempio 2 — Estremo infinito: \(\int_1^{+\infty}\frac{1}{x^2}\,dx\)

\(\displaystyle\lim_{c\to+\infty}\left[-\frac{1}{x}\right]_1^c = \lim_{c\to+\infty}\left(-\frac{1}{c}+1\right) = 1\)

→ Converge a \(1\).

Regola pratica — \(\int_1^{+\infty}\frac{1}{x^\alpha}\,dx\)

Converge se e solo se \(\alpha > 1\). Diverge per \(\alpha \leq 1\).
